###什么是HTTP？
全称为：超文本传输协议(Hyper Text Transfer Protocol)
两个电脑之间传输内容的协议,HTTP 的作用就是指导浏览器和服务器如何进行沟通。

###什么是curl？
>transfer a url

###HTTP的get请求示例：

用curl在gitbash上来创造一个请求：
curl -s -v -- https://www.baidu.com 得到请求结果如下：

脚本

Posted on 2018-04-25

1.什么是脚本

电脑的剧本。
如下所示：

cd ~/Desktop mkdir demo cd demo echo "hi" > index.htmlcd ~/Desktop
2.如何创建脚本

*创建一个文件
     `mkdir ~/local
      cd ~/local
      touch demo.txt`
*编辑文件
    `mkdir demo
     cd demo
     mkdir css js
     touch index.html css/style.css js/main.js
     exit`
*windows用户执行
    `sh ~/local/demo.txt`

即可运行此脚本了。

git简单入门

Posted on 2018-04-18 
在gitbash中配置好git后，
   1.创建一个目录作为项目目录 mkdir gitstore。
   2.进入这个目录 cd gitstore。
   3.运行 git init后目录里会多一个.git的仓库。
   4.在gitstore中任意添加文件 touch index.html。
   5.运行 git status -sb 会发现 index.html前面会有问号。
   6.运行 git add index.html 将该文件放入暂存区
   7.运行 git commit -m '第一次提交' 后放入本地仓库。
   8.在github上建一个新仓库如：gitstore(建立完后不要点击其他，直接创建)。
   9.点击ssh按钮后进行有本地仓库第二项逐条复制运行
   10.刷新页面本地仓库就已经上传到github仓库了。

简单linux命令的使用

Posted on 2018-04-18

##linux中ls的简单使用
list的缩写为ls，列表的意思。在命令行输入ls表示展现当前目录有哪些文件(不包括隐藏的文件)。
下面为ls的常用命令
1 ls -a
显示当前位置下的所有文件(包括以.开头的隐藏的文件)
2 ls -l
显示当前位置下的文件详细信息，文件是否有读写执行及文件创建的时间等功能。
3 ls -al
前两者的结合。

##linux中cat的简单使用
表示由第一行开始显示档案内容。

##linux中mv的简单使用
move的缩写为mv，在linux中表示为重命名的意思。如：mv 1.txt 2.txt 表示1.txt文件被重命名为2.txt

##linux中touch的使用
在linux中touch表示为新建文件的意思。如 touch 1.txt 则在当前目录下新建了一个1.txt的文件。

此文仅为简单介绍linux中个别常用命令，更多详情请看https://explainshell.com/explain?cmd=ls+-lrt
