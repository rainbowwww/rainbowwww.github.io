js里的数据类型

Posted on 2018-06-11

JS里的数据类型：

JS里有七中数据类型分别是：number、 string、 boolean、 symbol、 undefined、 null、 object。除object为复杂类型，其余皆为简单类型。
复杂类型由简单类型组成。

number

整数和小数（比如1和3.14）
表达方式有：

  • 整数和小数：1 1.1 .1
  • 科学记数法：1.23e2
  • 二进制：0b11
  • 八进制：011（后来 ES5 添加了 0o11 语法）
  • 十六进制：0x11

string

字符串就是零个或多个排在一起的字符，放在单引号或双引号之中。

区分空格字符串和空字符串

  • 空格字符串长度为1
  • 空字符串长度为0

单引号(双引号)字符串的内部，使用单引号(双引号)，须在内部的单引号(双引号)前面加上反斜杠，用来转义。

单引号字符串的内部，可以使用双引号。双引号字符串的内部，可以使用单引号。

boolean

boolean 的取值
只有两个值：true 和 false
a && b 在 a 和 b 都为 true 时，取值为 true；否则为 false
a || b 在 a 和 b 都为 false 时，取值为 false；否则为 true

null和undefined

这两个值都表示什么也没有，其实是语言设计的一个bug。
两者的区分：
声明的变量没有值或非对象不想给值是undefined 对象object没有进行给值就是null

object

object 就是几种基本类型（无序地）组合在一起 object 的 key 一律是字符串，不存在其他类型的 key
object 里面可以有 object 如果object里面的key不加引号则key的命名需要遵循标识符的命名规则。
如果key的命名符合标识符的命名规则，则object[‘key’]可以写成object.key 注意 object.key 与 object[key] 不同

    flex的相关知识

    flex container 的属性

  • flex-direction 方向
  • flex-wrap 换行
  • flex-flow 上面两个的简写
  • justify-content 主轴对齐方式(多余的空间的显示方式)
  • align-items 侧轴对齐方式（高度不一样时会体现出来）
  • align-content 多行/列内容对齐方式(控制行与行之间的空隙)
    遇到问题，查看是否固定了高度或宽度等查看出现的原因。
    flex item 的属性
  • flex-grow 增长比例(空间过多时)
  • flex-shrink 收缩比例(空间不够时)
  • flex-basis 默认大小
  • flex 上面三个的缩写
  • order 顺序代替双飞翼
  • align-self 自身的对齐方式

    div里面的div的居中方式

    1.若里面的div高度是确定的上下设置父元素padding，左右设置子元素margin：0 auto;
    2.子元素定宽定高可设置margin:auto;position:absolute; top:0; left:0; right:0;
    bottom:0;
    3.若子元素不定宽高设置flex，父元素display:flex; justify-content:center;
    align-item:center;
    详细可参考：方应杭的CSS居中的七种方法。

    文字的居中

    1.水平居中可用text-align:center;

什么是HTML？

>HTML（HyperText Markup Language，超文本标记语言） 是用来定义网页结构的一种描述语言。----MDN

#HTML发展史

* 1990年 Tim Berners-Lee 提出了超文本的概念。
* 1993年 IETF于正式开始制定 HTML规范。
* 1995年 发布了HTML2.0 版本。
* 1994年 Berners-Lee为了Web 发展而成立了 W3C。
* 1997年 w3c发布了HTML3.2推荐标准。
* 2000年 HTML4.0成为ISO标准。
* 2014年 W3C和WHATWG发布了HTML5标准的最终版。

   10.刷新页面本地仓库就已经上传到github仓库了。

##linux中ls的简单使用
list的缩写为ls，列表的意思。在命令行输入ls表示展现当前目录有哪些文件(不包括隐藏的文件)。
下面为ls的常用命令
1 ls -a
显示当前位置下的所有文件(包括以.开头的隐藏的文件)
2 ls -l
显示当前位置下的文件详细信息，文件是否有读写执行及文件创建的时间等功能。
3 ls -al
前两者的结合。

##linux中cat的简单使用
表示由第一行开始显示档案内容。

##linux中mv的简单使用
move的缩写为mv，在linux中表示为重命名的意思。如：mv 1.txt 2.txt 表示1.txt文件被重命名为2.txt

##linux中touch的使用
在linux中touch表示为新建文件的意思。如 touch 1.txt 则在当前目录下新建了一个1.txt的文件。
