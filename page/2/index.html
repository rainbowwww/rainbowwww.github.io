HTML

Posted on 2018-05-02

什么是HTML？

>HTML（HyperText Markup Language，超文本标记语言） 是用来定义网页结构的一种描述语言。----MDN

#HTML发展史

* 1990年 Tim Berners-Lee 提出了超文本的概念。
* 1993年 IETF于正式开始制定 HTML规范。
* 1995年 发布了HTML2.0 版本。
* 1994年 Berners-Lee为了Web 发展而成立了 W3C。
* 1997年 w3c发布了HTML3.2推荐标准。
* 2000年 HTML4.0成为ISO标准。
* 2014年 W3C和WHATWG发布了HTML5标准的最终版。

HTML有哪些标签？

| Element|Description |
|:------:|:----------:|
|<html>  | 代表 HTML 或 XHTML 文档的根。其他所有元素必须是这个元素的子节点。|
|<head>  |代表关于文档元数据的一个集合，包括脚本或样式表的链接或内容。|
|<title> |定义文档的标题，将显示在浏览器的标题栏或标签页上。该元素只能包含文本，包含的标签不会被解释。|
|<base>  |定义页面上相对 URL 的基准 URL。|
|<link>  |用于链接外部的 CSS 到该文档。|
|<meta>  |定义其他 HTML 元素无法描述的元数据。|
|<style> |用于内联 CSS。|
|<script>|定义一个内联脚本或链接到外部脚本。脚本语言是 JavaScript。|
|<noscript>|定义当浏览器不支持脚本时显示的替代文字。|
|<body>|代表 HTML 文档的内容。在文档中只能有一个 <body> 元素。|
|<section>|定义文档中的一个章节。|
|<nav>|定义只包含导航链接的章节。|
|<h1>,<h2>,<h3>,<h4>,<h5>,<h6>|    标题元素实现了六层文档标题，<h1> 是最大的标题，<h6> 是最小的标题。标题元素简要地描述章节的主题。|
|<header>|定义页面或章节的头部。它经常包含 logo、页面标题和导航性的目录。|
|<footer>|定义页面或章节的尾部。它经常包含版权信息、法律信息链接和反馈建议用的地址。|
|<main>|定义文档中主要或重要的内容。|
|……|……|

脚本

Posted on 2018-04-25

1.什么是脚本

电脑的剧本。
如下所示：

cd ~/Desktop mkdir demo cd demo echo "hi" > index.htmlcd ~/Desktop
2.如何创建脚本

*创建一个文件
     `mkdir ~/local
      cd ~/local
      touch demo.txt`
*编辑文件
    `mkdir demo
     cd demo
     mkdir css js
     touch index.html css/style.css js/main.js
     exit`
*windows用户执行
    `sh ~/local/demo.txt`

即可运行此脚本了。

git简单入门

Posted on 2018-04-18 
在gitbash中配置好git后，
   1.创建一个目录作为项目目录 mkdir gitstore。
   2.进入这个目录 cd gitstore。
   3.运行 git init后目录里会多一个.git的仓库。
   4.在gitstore中任意添加文件 touch index.html。
   5.运行 git status -sb 会发现 index.html前面会有问号。
   6.运行 git add index.html 将该文件放入暂存区
   7.运行 git commit -m '第一次提交' 后放入本地仓库。
   8.在github上建一个新仓库如：gitstore(建立完后不要点击其他，直接创建)。
   9.点击ssh按钮后进行有本地仓库第二项逐条复制运行
   10.刷新页面本地仓库就已经上传到github仓库了。

简单linux命令的使用

Posted on 2018-04-18

##linux中ls的简单使用
list的缩写为ls，列表的意思。在命令行输入ls表示展现当前目录有哪些文件(不包括隐藏的文件)。
下面为ls的常用命令
1 ls -a
显示当前位置下的所有文件(包括以.开头的隐藏的文件)
2 ls -l
显示当前位置下的文件详细信息，文件是否有读写执行及文件创建的时间等功能。
3 ls -al
前两者的结合。

##linux中cat的简单使用
表示由第一行开始显示档案内容。

##linux中mv的简单使用
move的缩写为mv，在linux中表示为重命名的意思。如：mv 1.txt 2.txt 表示1.txt文件被重命名为2.txt

##linux中touch的使用
在linux中touch表示为新建文件的意思。如 touch 1.txt 则在当前目录下新建了一个1.txt的文件。
