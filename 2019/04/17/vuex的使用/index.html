vuex的使用

Posted on 2019-04-17

vuex有用的核心就是store（仓库）。对仓库里面的数据进行处理等操作。

创建的方法

安装vuex 之后，创建一个store并在模块化的构建系统中，在开头一句引用了Vue.use(Vuex)。

各个对象的用途

  1. state ：初始化数据，唯一的数据源。

  2. getter：可以认为是store的计算属性。

  3. mutations： 对数据的事件处理，处理同步事件

  4. actions：提交的是mutation，而不是直接变更状态。 mutations相似区别在于可以包含异步事件。

在组件中的调用方法

  1. state在组件中的调用

    在组件中的conputed用…mapstate{[ ‘数据’]}来进行引用

  2. getter在组件的使用

    在组件中的conputed用…mapGetters{[ ‘数据’]}来进行引用

  3. mutations在组件中的使用

  4. actions在组件中的应用
