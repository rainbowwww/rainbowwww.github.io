网页布局

Posted on 2018-05-23

布局流程分为两种

  1. 需要支持IE8用float布局定宽。
  2. 不需要支持IE8用flex布局弹性宽度。
    基本原则：
  • 不到万不得已，不要写死width和height
  • 尽量用高级语法，如calc、flex
  • 如果是IE，就全部写死

    float布局套路：

  1. 给儿子全加float:left（right）

  2. 老子加.clearfix

  • 老子加 display: flex

  • 老子加 justify-content: space-between;

    flex的相关知识

    flex container 的属性

  • flex-direction 方向
  • flex-wrap 换行
  • flex-flow 上面两个的简写
  • justify-content 主轴对齐方式(多余的空间的显示方式)
  • align-items 侧轴对齐方式（高度不一样时会体现出来）
  • align-content 多行/列内容对齐方式(控制行与行之间的空隙)
    遇到问题，查看是否固定了高度或宽度等查看出现的原因。
    flex item 的属性
  • flex-grow 增长比例(空间过多时)
  • flex-shrink 收缩比例(空间不够时)
  • flex-basis 默认大小
  • flex 上面三个的缩写
  • order 顺序代替双飞翼
  • align-self 自身的对齐方式

    div里面的div的居中方式

    1.若里面的div高度是确定的上下设置父元素padding，左右设置子元素margin：0 auto;
    2.子元素定宽定高可设置margin:auto;position:absolute; top:0; left:0; right:0;
    bottom:0;
    3.若子元素不定宽高设置flex，父元素display:flex; justify-content:center;
    align-item:center;
    详细可参考：方应杭的CSS居中的七种方法。

    文字的居中

    1.水平居中可用text-align:center;
