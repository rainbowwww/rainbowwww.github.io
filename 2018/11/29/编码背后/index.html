读《编码》总结一

因读了该书的三分之一，越往后读，越记不住前面的知识点，前后知识点关联脱节，于是就诞生这篇总结博客。

与门

只有当两个继电器都被触发的时候灯泡才会亮。这样两个继电器的串联被称为一个”与门”

电器工程师用如下专门的符号表示一个与门

如果我们将开关闭合为1，开启为1，那么与门的输入与输出之间的关系可以用该表格表示：

And 0 1
0 0 0
1 0 1

或门

并联电路，两个继电器的输出是接在一起的，这个连在一起的输出为灯泡提供了电源。任何一个继电器都能点亮灯泡。当上面的开关或下面的开关闭合，灯泡都会发光，这里的关键词是或，因此这样的门被称为“或门”

电器工程师用如下专门的符号表示一个或门

如果我们将开关闭合为1，开启为1，那么或门的输入与输出之间的关系可以用该表格表示：

or 0 1
0 0 1
1 1 1

反向器

开关断开时，灯泡被点亮。开关闭合时，灯泡就会熄灭。以这种方式连接的继电器叫反向器。反向器不是逻辑门（一个逻辑门通常有两个或多个输入）如下符号表示反向器:

或非门的输出如下：

0 1 0
1 0 0
